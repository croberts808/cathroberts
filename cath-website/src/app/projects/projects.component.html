Flower Shop

Last December, I started a Google course through Coursera to learn more about UX Design and grow my skills as a designer and researcher. . . .

There are 7 courses total, and I am currently working my way through the 4th, “Conduct UX Research and Test Early Concepts.” In the first 3 courses, I learned all about user personas, user pain points, wireframing, prototyping, empathy, accessibility, and how to create user journeys and competitive audits.

In order to work through these courses, it was necessary to come up with an idea for a mobile or web application to put my newfound knowledge to work. I decided to create a floral delivery app that solves user pain points and is simple to use. Through the class, I learned how to use Figma to first create wireframes for the app, and then turn those wireframes into a prototype. See below for a walkthrough of my little app! What I Learned:

  • How to create digital wireframes from paper
  • How to use the Figma components tool
  • How to use Figma Variants to create interactive components
My Favorite Parts of the Entire Process:
  • Creating User Personas (see 'Alicia Wren' above)
  • Creating a User Journey Map to anticipate potential needs for the app
  • Learning how to navigate and use Figma
